Project 1 — Grameenphone Ltd. Analytics Dataset
Monthly Analytics-Ready Dataset · 2020–2024

Built a clean, monthly analytics-ready dataset for Grameenphone Ltd. by combining stock price data from Yahoo Finance, firm-level fundamentals, and macroeconomic indicators from the World Bank API. The final dataset contains 60 monthly rows across 12 variables, spanning the 2020–2024 period.

  • Data sources: Yahoo Finance (stock data), firm-level variables, World Bank API (macro indicators)

  • Quality checks: Missing values treated and a full data dictionary documented for all 12 variables

Project 3 — NVDA ARIMA vs LSTM Return Forecasting
Model Comparison · Chronological 70/15/15 Split

Compared an ARIMA model against an LSTM neural network for forecasting NVDA daily log returns, using a chronological 70/15/15 train/validation/test split. The selected model, ARIMA(1,0,0), achieved an AIC of −4208.7063.
